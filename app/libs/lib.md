# FlowBot 企业微信自动化核心依赖库

> FlowBot 是一款基于 **Android 无障碍服务** 的**企业微信RPA机器人**，通过调用以下核心依赖库实现**企业微信自动化**操作，支持**企业微信消息收发**、**企业微信好友管理**、**企业微信群管理**、**企业微信朋友圈发布**等功能。

## 核心依赖库

### 1. Rhino JavaScript 引擎

**文件**: `rhino-1.7.14-jdk7.jar` / `rhino-1.7.14-jdk7-sources.jar`

**用途**: 企业微信自动化脚本执行引擎

**企业微信应用场景**:
- 执行企业微信自动化脚本
- 处理企业微信消息解析逻辑
- 运行企业微信RPA业务流程
- 企业微信机器人指令调度

**技术说明**:
Rhino 是 Mozilla 开发的 JavaScript 引擎，FlowBot 使用 Rhino 作为**企业微信自动化脚本**的运行环境。开发者可以使用 JavaScript 编写**企业微信机器人**的自动化逻辑，包括：
- 企业微信消息自动回复
- 企业微信群发消息
- 企业微信好友自动添加
- 企业微信群成员管理
- 企业微信朋友圈定时发布

```javascript
// 企业微信自动化脚本示例
// 发送企业微信文本消息
sendTextMessage("企业微信好友昵称", "这是一条企业微信自动发送的消息");

// 企业微信群发消息
groupSendMessage("企业微信群名称", "企业微信群发内容");

// 企业微信添加好友
addFriend("手机号码", "企业微信好友验证消息");
```

### 2. DEX 编译工具

**文件**: `dx.jar`

**用途**: Android DEX 字节码编译

**企业微信应用场景**:
- 编译企业微信自动化脚本为DEX
- 企业微信机器人插件热加载
- 企业微信RPA模块动态编译

**技术说明**:
dx.jar 是 Android SDK 提供的 DEX 编译工具，FlowBot 使用它将**企业微信自动化脚本**编译为 Android 可执行的 DEX 字节码，实现：
- 企业微信机器人脚本的即时编译执行
- 企业微信RPA插件的动态加载
- 企业微信自动化模块的热更新

### 示例代码

#### 企业微信自动回复机器人
```javascript
// 企业微信自动回复脚本
// 基于 Rhino 引擎执行

// 监听企业微信消息
onMessage(function(message) {
    // 获取企业微信消息内容
    var content = message.getContent();
    var sender = message.getSender();
    
    // 企业微信关键词自动回复
    if (content.indexOf("价格") !== -1) {
        sendTextMessage(sender, "您好，这是企业微信自动回复：请查看价格表...");
    }
    
    if (content.indexOf("咨询") !== -1) {
        sendTextMessage(sender, "您好，这是企业微信自动回复：请问有什么可以帮您？");
    }
});

// 企业微信定时群发
setInterval(function() {
    var groups = getGroupList();  // 获取企业微信群列表
    groups.forEach(function(group) {
        sendGroupMessage(group.name, "企业微信定时群发消息内容");
    });
}, 3600000);  // 每小时执行一次企业微信群发
```

#### 企业微信好友自动添加
```javascript
// 企业微信自动添加好友脚本

// 从手机号列表批量添加企业微信好友
var phones = ["13800138001", "13800138002", "13800138003"];

phones.forEach(function(phone) {
    // 调用企业微信添加好友接口
    addFriend(phone, "您好，我是企业微信机器人自动添加的好友");
    
    // 企业微信添加好友间隔，避免频繁操作
    sleep(5000);
});

// 企业微信自动通过好友请求
onFriendRequest(function(request) {
    // 自动通过企业微信好友申请
    acceptFriend(request);
    
    // 发送企业微信欢迎消息
    sendTextMessage(request.getNickname(), "欢迎添加企业微信好友，我是智能客服机器人！");
});
```

## 企业微信自动化最佳实践

### 1. 企业微信消息发送频率控制
- 单条企业微信消息发送间隔建议 > 1秒
- 企业微信群发消息建议分批次发送
- 避免短时间内大量发送企业微信消息触发风控

### 2. 企业微信好友添加策略
- 单日企业微信添加好友数量建议 < 50人
- 企业微信好友验证消息要真实可信
- 添加企业微信好友后及时发送问候消息

### 3. 企业微信群管理规范
- 企业微信群名称要规范命名
- 企业微信群公告要及时更新
- 企业微信群成员要定期清理

### 4. 企业微信朋友圈发布技巧
- 企业微信朋友圈发布时间选择用户活跃时段
- 企业微信朋友圈内容要真实有价值
- 企业微信朋友圈配图要清晰美观
