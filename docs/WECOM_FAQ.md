## 企业微信基础问题

### Q1: FlowBot 是什么？

**A:** FlowBot 是一款基于 **Android 无障碍服务** 的**企业微信RPA机器人**，通过API控制**企业微信**实现自动化运营。支持**企业微信消息收发**、**企业微信好友管理**、**企业微信群管理**、**企业微信朋友圈发布**等功能。

### Q2: FlowBot 支持哪些企业微信版本？

**A:** 目前支持以下**企业微信**版本：
- 企业微信 4.1.32（推荐）
- 企业微信 4.1.30
- 企业微信 4.1.28
- 企业微信 4.1.26

建议使用已兼容版本中的最新版本。

### Q3: 企业微信机器人需要Root权限吗？

**A:** 不需要。FlowBot 使用 **Android 官方无障碍服务**，无需破解Root权限，兼容99%安卓机型。

### Q4: 企业微信机器人会被封号吗？

**A:** FlowBot 采用**企业微信官方SDK**和Android官方无障碍服务，**无HOOK、无侵入、无内存修改**，正常使用不会导致企业微信封号。但请遵守企业微信使用规范，避免频繁操作。

## 企业微信机器人配置

### Q5: 如何获取企业微信机器人 robotId？

**A:** 
1. 下载并安装 FlowBot APP
2. 打开APP，按照提示绑定企业微信账号
3. 系统会自动分配一个 robotId
4. 在API调用时使用该 robotId

### Q6: 企业微信机器人如何保持长时间运行？

**A:** 
1. 手机保持充电状态
2. 关闭企业微信省电模式
3. 授予 FlowBot 后台运行权限
4. 避免清理后台进程
5. 使用稳定的网络连接

### Q7: 企业微信机器人可以同时登录多个账号吗？

**A:** 每个 robotId 对应一个企业微信账号。如需管理多个企业微信账号，需要申请多个 robotId。

### Q8: 企业微信机器人支持哪些安卓机型？

**A:** FlowBot 兼容99%安卓机型，包括：
- 华为、小米、OPPO、vivo
- 三星、一加、魅族
- 荣耀、realme、iQOO
- 其他主流安卓品牌

## 企业微信消息相关

### Q9: 企业微信消息发送失败怎么办？

**A:** 请按以下步骤排查：
1. 检查企业微信机器人是否在线
2. 检查企业微信接收人是否存在
3. 检查企业微信消息内容是否合规
4. 检查企业微信发送频率是否过高
5. 检查网络连接是否正常

### Q10: 企业微信消息发送有频率限制吗？

**A:** 是的，建议：
- 单条企业微信消息发送间隔 > 1秒
- 企业微信群发消息分批次发送
- 避免短时间内大量发送企业微信消息

### Q11: 企业微信支持发送哪些消息类型？

**A:** 支持以下**企业微信消息类型**：
- 企业微信文本消息
- 企业微信图片消息
- 企业微信文件消息
- 企业微信视频消息
- 企业微信名片消息
- 企业微信链接消息
- 企业微信小程序消息

### Q12: 企业微信消息可以撤回吗？

**A:** 企业微信消息撤回功能为定制功能，需要联系官方客服申请开通。

### Q13: 如何实现企业微信自动回复？

**A:** 企业微信自动回复配置示例：
```javascript
// 企业微信自动回复规则配置
const autoReplyRules = {
    '价格': '企业微信自动回复：请查看价格表...',
    '咨询': '企业微信自动回复：请问有什么可以帮您？',
    '人工': '企业微信自动回复：正在为您转接人工客服...',
};

// 企业微信消息监听
wecomAPI.onMessage(async (message) => {
    const reply = findReply(message.content, autoReplyRules);
    if (reply) {
        await wecomAPI.sendText({
            robot_id: 'YOUR_ROBOT_ID',
            receiver: message.sender,
            content: reply,
        });
    }
});
```

## 企业微信好友管理

### Q14: 企业微信添加好友有数量限制吗？

**A:** 企业微信对好友添加有频率限制：
- 每日添加好友数量建议 < 50人
- 每次添加间隔建议 > 10秒
- 使用真实可信的企业微信验证消息

### Q15: 企业微信添加好友失败的原因？

**A:** 可能原因：
1. 对方企业微信设置了隐私权限
2. 企业微信添加频率过高被限制
3. 手机号未注册企业微信
4. 企业微信验证消息不合规
5. 网络连接问题

### Q16: 如何检测企业微信单向好友？

**A:** 使用企业微信单向好友检测API：
```javascript
// 企业微信单向好友检测
const oneWayFriends = await wecomAPI.checkOneWayFriends({
    robot_id: 'YOUR_ROBOT_ID',
});

console.log('企业微信单向好友数量:', oneWayFriends.length);
```

### Q17: 企业微信好友信息可以修改吗？

**A:** 可以修改企业微信好友备注，但不能修改好友的其他信息。

### Q18: 企业微信如何批量添加好友？

**A:** 企业微信批量添加好友示例：
```javascript
// 企业微信批量添加好友
const phones = ['13800138001', '13800138002', '13800138003'];

for (const phone of phones) {
    await wecomAPI.addFriend({
        robot_id: 'YOUR_ROBOT_ID',
        phone: phone,
        verify_msg: '企业微信好友申请',
    });
    
    // 企业微信添加间隔
    await sleep(10000);
}
```

## 企业微信安全与合规

### Q27: FlowBot 是否安全合规？

**A:** 是的。FlowBot 采用：
- **企业微信官方SDK** - 基于官方API
- **Android官方无障碍服务** - 政府支持的自动化方案
- **无HOOK、无侵入** - 不修改企业微信客户端
- **工信部政策支持** - 无障碍服务改造

参考：[工信部无障碍服务政策](https://www.gov.cn/gongbao/content/2020/content_5570087.htm)

### Q28: 企业微信数据是否安全？

**A:** FlowBot 采用本地运行模式，企业微信数据不上传服务器。如需更高级数据安全方案，可申请企业定制版。

### Q29: 哪些行为会导致企业微信封号？

**A:** 以下行为可能导致企业微信封号：
1. 使用非官方HOOK注入工具
2. 频繁发送垃圾消息
3. 批量添加好友被举报
4. 发送违规内容
5. 违反企业微信使用规范

### Q30: 企业微信机器人可以用于营销吗？

**A:** 可以用于合规的企业微信营销，但请注意：
1. 不发送垃圾消息
2. 不骚扰企业微信用户
3. 遵守企业微信营销规范
4. 提供退订选项

## 企业微信API调用

### Q31: 企业微信API如何调用？

**A:** 企业微信API调用示例：
```javascript
// 企业微信API基础配置
const config = {
    robotId: 'YOUR_ROBOT_ID',
    baseUrl: 'https://api.flowbot.cn',
};

// 企业微信发送消息API
async function sendMessage(receiver, content) {
    const response = await fetch(`${baseUrl}/api/v1/wecom/send`, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${config.robotId}`,
        },
        body: JSON.stringify({
            receiver: receiver,
            content: content,
        }),
    });
    return response.json();
}
```

### Q32: 企业微信API返回错误码怎么办？

**A:** 常见企业微信API错误码：
| 错误码 | 说明 | 解决方案 |
|--------|------|----------|
| 1001 | 企业微信机器人不存在 | 检查 robotId |
| 1002 | 企业微信机器人离线 | 启动机器人APP |
| 2001 | 企业微信接收人不存在 | 检查接收人昵称 |
| 2002 | 企业微信消息发送失败 | 检查网络连接 |
| 3001 | 企业微信好友添加失败 | 降低添加频率 |

### Q33: 企业微信API支持哪些编程语言？

**A:** 企业微信API支持所有能发送HTTP请求的编程语言：
- JavaScript/Node.js

- Java
- PHP
- Go
- C#
- Ruby
- Shell/curl

### Q34: 企业微信API有调用频率限制吗？

**A:** 企业微信API调用建议：
- 单个API每秒调用 < 10次
- 消息发送API每秒 < 1次
- 避免并发调用同一API
